# About Eli Gault-Crabb

A plain HTML/CSS/JavaScript personal portfolio website with a cinematic ocean background, typewriter banner, taped Polaroid navigation, journal-style modals, and simple branch pages.

## Open Locally

Open `index.html` directly in a browser.

No build step, package install, framework, or local server is required.
